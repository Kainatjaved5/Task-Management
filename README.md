## üìñ Usage Guide

### Creating Tasks
1. Click the **"Add Task"** button in the Tasks section
2. Fill in task details:
   - **Title** (required)
   - **Description** (optional)
   - **Due Date** (optional)
   - **Priority** (Low, Medium, High)
   - **Category** (optional)
   - **Status** (To Do, In Progress, Completed)
3. Click **"Save Task"** to create

### Managing Tasks
- **Edit**: Click the edit icon on any task card
- **Delete**: Click the trash icon (with confirmation)
- **Change Status**: Click the exchange icon to cycle through statuses
- **View Details**: All task information is displayed on the task cards

### Using Categories
1. Navigate to the **Categories** section
2. Click **"Add Category"** to create new categories
3. Choose a custom color for each category
4. Delete categories by clicking on them (tasks remain unaffected)

### Search & Filter
- **Search**: Type in the search box to find tasks by title or description
- **Status Filter**: Filter by To Do, In Progress, or Completed
- **Priority Filter**: Filter by Low, Medium, or High priority
- **Category Filter**: Filter by specific categories

### Dashboard Overview
- **Statistics**: View total, completed, pending, and in-progress tasks
- **Progress Bar**: Visual representation of overall completion
- **Recent Tasks**: Quick view of the 5 most recently created tasks

### Data Management
- **Export Data**: Download all tasks and categories as JSON backup
- **Clear Data**: Remove all data (requires double confirmation)
